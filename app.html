FIZZ BUZZ: A Coding Project

When it counts- you Fizz Buzz. A javascript coding exercise using loops. The program that prints numbers to the page. For multiples of '3' displays 'Fizz' instead of the number. For multiples of '5' displays 'Buzz'. Finally, if the number is a multiple of both- displays 'FizzBuzz'.

Instructions

Enter Fizz, Buzz and range values in the 0 - 100 range. Higher Range value needs to be smaller than or equal to Lower Range value. Press the button and the values will appear.
